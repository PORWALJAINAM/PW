1. html - The root element of an HTML page. All other elements must be nested within it.
  2. head - Contains metadata about the document, such as the title and links to stylesheets.
  3. title - Specifies the title of the document, which appears in the browser's title bar.
  4. body- Contains the main content of the document, such as text, images, and other elements.
  5. h1 to h6- Heading tags used to create different levels of headings. <h1> is the largest and <h6> is the smallest.
  6. p - Used to create paragraphs of text.
